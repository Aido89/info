# 🚀 Инструкция по настройке проекта

## Что нужно сделать перед запуском:

### 1. ✅ Добавить изображения

Создайте папку `images` и добавьте туда:

#### Аватар (обязательно):
- **Файл**: `images/avatar.jpg` или `images/avatar.png`
- **Размер**: минимум 400x400px (рекомендуется квадратное)
- **Формат**: JPG или PNG

#### QR-коды (опционально, но рекомендуется):
- `images/qr-whatsapp.jpg` - QR-код для WhatsApp
- `images/qr-telegram.jpg` - QR-код для Telegram
- `images/qr-instagram.jpg` - QR-код для Instagram
- `images/qr-youtube.jpg` - QR-код для YouTube
- `images/qr-tiktok.jpg` - QR-код для TikTok

**Как создать QR-коды:**
1. Перейдите на https://www.qr-code-generator.com/
2. Введите ссылку на ваш профиль
3. Скачайте изображение
4. Сохраните в папку `images/` с правильным именем

### 2. ⚙️ Настроить API ключ (опционально)

Для работы AI-консультанта:

1. Получите API ключ Gemini: https://makersuite.google.com/app/apikey
2. Откройте `script.js`
3. Найдите строку: `const apiKey = "";`
4. Вставьте ваш ключ: `const apiKey = "ваш_ключ_здесь";`

**Без API ключа:** AI-консультант не будет работать, но остальной сайт работает нормально.

### 3. 📝 Настроить контакты (опционально)

Проверьте все ссылки в секции контактов (`index.html`, секция `#contacts`):
- WhatsApp
- Telegram
- Email
- YouTube
- Instagram
- TikTok
- GitHub

### 4. 🎨 Настроить цвета (опционально)

Если хотите изменить цветовую схему, откройте `styles.css` и измените:
```css
.accent-bg {
    background-color: #0d9488; /* Ваш цвет */
}
```

## 🚀 Запуск:

### Вариант 1: Простой (рекомендуется)
```bash
python -m http.server 8080
```
Затем откройте: http://localhost:8080

### Вариант 2: С npx
```bash
npx serve
```

### Вариант 3: Прямое открытие
Просто дважды кликните на `index.html`

## 🐛 Решение проблем:

### Изображения не загружаются:
- Проверьте, что файлы находятся в папке `images/`
- Проверьте правильность имен файлов
- Проверьте формат файлов (JPG, PNG)

### AI-консультант не работает:
- Проверьте, что API ключ вставлен в `script.js`
- Проверьте интернет-соединение
- Проверьте консоль браузера на ошибки (F12)

### Стили не применяются:
- Убедитесь, что файл `styles.css` находится в той же папке
- Проверьте консоль браузера на ошибки загрузки

## 📞 Нужна помощь?

Если что-то не работает, проверьте:
1. Консоль браузера (F12 → Console)
2. Вкладку Network (F12 → Network) для проверки загрузки файлов
3. Убедитесь, что все файлы на месте
