# Портфолио Айдына Каканова

Современный, отзывчивый веб-сайт-портфолио для IT-специалиста, архитектора цифровых решений и технологического продюсера.

## 🚀 Особенности

- ✨ **Современный дизайн** с плавными анимациями и переходами
- 📱 **Полностью адаптивный** - отлично работает на всех устройствах
- 🎨 **Интерактивные элементы** с hover-эффектами
- 🤖 **AI-консультант** на базе Gemini API
- ⚡ **Оптимизированная производительность** с lazy loading
- ♿ **Доступность** - поддержка клавиатурной навигации и screen readers
- 🎯 **SEO-оптимизация** с мета-тегами

## 📁 Структура проекта

```
.
├── index.html          # Основной HTML файл
├── styles.css          # Все стили и анимации
├── script.js           # JavaScript функциональность
├── README.md           # Документация

```

## 🚀 Быстрый старт

1. **Клонируйте или скачайте проект**
   ```bash
   git clone <repository-url>
   cd info
   ```

2. **Настройте API ключ (опционально)**
   - Откройте `script.js`
   - Найдите переменную `apiKey`
   - Вставьте ваш Gemini API ключ

3. **Откройте в браузере**
   - Просто откройте `index.html` в браузере
   - Или используйте локальный сервер:
     ```bash
     # Python
     python -m http.server 8000
     
     # Node.js
     npx serve
     ```

## 📝 Настройка

### API ключ Gemini

Для работы AI-консультанта необходимо:

1. Получить API ключ от Google Gemini
2. Добавить его в `script.js`:
   ```javascript
   const apiKey = "YOUR_API_KEY_HERE";
   ```

### Изменение цветовой схемы

Основной цвет (teal) можно изменить в `styles.css`:
```css
.accent-bg {
    background-color: #0d9488; /* Измените на нужный цвет */
}
```

### Добавление QR-кодов

Поместите изображения QR-кодов в корневую папку проекта и обновите пути в `index.html`.
